A flutter client library for the [IPFS HTTP API](https://docs.ipfs.io/reference/http/api/).

### Features

* Make a directory
* List all directories/files in a directory/subdirectory
* File status
* Write a file
* Read a file
* Remove a file/directory
* Move/Rename a file

### Usage
```
  IpfsClient ipfsClient = IpfsClient();
  ot
```
